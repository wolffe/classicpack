# ClassicPack

[Official Website](https://getbutterfly.com/classicpress-plugins/classicpack/)

[More ClassicPress Plugins](https://getbutterfly.com/classicpress-plugins/)

**ClassicPack** is a modular toolkit for [ClassicPress](https://www.classicpress.net/). Turn features on or off from a single **ClassicPack** admin screen—no bloat from code you do not use.

Included modules cover redirects, email to commenters, auto-saving remote images, safer post deletion with attachments, users online, user list extras, post type switching (Classic Editor), duplicate post, and more.

## Requirements

- ClassicPress 2.5+ or WordPress 6.2+
- PHP 8.5+
