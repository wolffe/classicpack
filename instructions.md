# ClassicPack

Modular features for **ClassicPress 2.5+** and **WordPress 6.9+**. Enable only what you need under **ClassicPack → Modules**.

## Installation

1. Upload the `classicpack` folder to `wp-content/plugins/`.
2. Activate **ClassicPack** from the Plugins screen.
3. Open **ClassicPack** in the admin sidebar, then **Modules**, and enable features by category (**Performance & SEO**, **Media**, **Users**).
4. Use the left-hand **ClassicPack** menu to open each enabled module’s screen.

## Modules

- **Core Redirects Manager** — Lists and removes `_wp_old_slug` redirects stored in post meta.
- **Email Commenters** — Sends a one-time email to approved commenters on a chosen post.
- **Auto Save Images** — Downloads remote images into the Media Library when you save a post; optional featured image and per-post skip.
- **Users Online** — Tracks visitors; a summary is shown in the Dashboard widget (when enabled and your role can list users).
- **User Manager** — Adds Registration Date and Last Login on the Users screen; optional “members only” flag per post/page; login link uses your site’s normal login URL.

## Requirements

- **PHP** 8.5 or newer  
- **ClassicPress** 2.5+ *or* **WordPress** 6.9+
